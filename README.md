# EOSYN.NET

Personal portfolio site with markdown content, site bookmarks, and custom themes.

## Setup

```bash
npm install
npm run build
npx serve .
```

## Structure

```
├── config/          # Settings and site lists
├── content/         # Markdown files
├── pages/           # HTML pages
├── assets/          # Scripts and styles
└── data/            # Generated JSON
```

## Deploy

Push to GitHub, enable Pages with GitHub Actions as source.
